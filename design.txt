Http Service
============
Endpoints:
GET /files - list all files in a tree structure
POST /search - search across all file names/directories
POST /backup - backup selected files/directories

File Index Service
==================
ListAll - returns a list of files in a tree structure
Search(pattern) - returns a list of files matching pattern
NotifyNew(file) - notify that a new file has been created
NotifyChanged(file) - notify that a file has changed
NotifyExisting(file) - notify that file already exists (on startup)

Backup Service
==============
Start() - backup file/directory

Timer service
